About: You can cause the wearable to start an activity by pressing a button in the companion app UI. You can also take a picture on the companion device, which will be transmitted as an Asset for display on the wearable.

How to run: Install the Application APK on the companion, and the Wearable APK on the wearable. Start the sample by running the DataLayer application on the companion.

Wearable APIs used: DataApi, MessageApi, NodeApi, WearableListenerService
